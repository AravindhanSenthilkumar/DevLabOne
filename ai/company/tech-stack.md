# Tech Stack

## Frontend
- Angular for web applications.
- TypeScript for application code.
- RxJS for reactive state and async workflows.
- Angular Material or a consistent design system for common UI controls.

## Backend
- .NET for core APIs and enterprise-grade services.
- Node.js for lightweight services, automation, and developer tooling.
- REST as the default API style, with OpenAPI documentation.

## Database
- PostgreSQL as the default relational database.
- Redis for caching and short-lived coordination when needed.
- Object storage for files and documents.

## Cloud and Infrastructure
- Cloud provider: Azure or AWS depending on customer and deployment needs.
- Docker for local and deployable service packaging.
- CI/CD for build, test, security checks, and deployment.
- Infrastructure as Code for repeatable environments.

## AI and Automation
- AI agents for planning, development, testing, review, documentation, and operations support.
- Retrieval from company docs and codebase before generating recommendations.
- Human approval required for production deployment, customer communication, and security exceptions.

## Observability
- Centralized logging.
- Metrics for latency, error rate, throughput, and cost.
- Alerts for production failures and security events.
