# Glossary

## AI Agent
A specialized assistant responsible for a role such as planning, coding, testing, security, or documentation.

## Acceptance Criteria
Specific conditions that must be true before work is considered complete.

## Definition of Done
The shared checklist that determines whether work is complete and releasable.

## Feature
A user-facing or business-facing capability that delivers value.

## Hotfix
An urgent production fix for a serious issue.

## Pull Request
A proposed code change submitted for review before merging.

## Release
A packaged set of changes delivered to an environment or users.
