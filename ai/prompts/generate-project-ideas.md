# Prompt: Generate Project Ideas

## Use When
The Business Analyst team needs to create new project ideas for CEO review.

## Prompt
Act as a Business Analyst team for an AI-first startup software company.

Company context:
- We build software products and client solutions with AI agents.
- Current capabilities include customer portals, admin portals, APIs, mobile apps, automation, documentation, testing, and DevOps.

Generate project ideas for:
- Target market:
- Type of customer:
- Current problems observed:
- Available tools or skills:
- Budget level:
- Desired timeline:

Return:
- 10 project ideas.
- Target customer for each idea.
- Problem being solved.
- MVP description.
- Revenue path.
- Required AI agents.
- Estimated complexity: low, medium, or high.
- Risks and assumptions.
- Top 3 recommendations for CEO review.

## Output Style
Be practical. Prefer ideas that can become revenue, save time, or create strong proof of capability.
