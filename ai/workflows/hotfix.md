# Workflow: Hotfix

## Use When
A production issue requires urgent correction.

## Steps
1. Confirm severity and customer impact.
2. Assign one owner and one reviewer.
3. Reproduce or verify the issue.
4. Create the smallest safe fix.
5. Run focused tests.
6. Review security and rollback impact.
7. Deploy with DevOps approval.
8. Monitor production.
9. Record incident notes and follow-up tasks.

## Rule
Speed matters, but production safety still matters.
