# Workflow: Feature Development

## Steps
1. CEO or Product owner confirms business goal.
2. Business Analyst writes user stories and acceptance criteria.
3. UX Designer drafts user flow and UI expectations.
4. Solution Architect defines technical approach.
5. Frontend, Backend, and Database agents implement.
6. QA creates and runs test plan.
7. Security reviews risk areas.
8. Documentation updates user and developer docs.
9. Project Manager confirms definition of done.
10. DevOps deploys through the release process.

## Required Outputs
- User story.
- Technical plan.
- Tests.
- Documentation updates.
- Release note.
