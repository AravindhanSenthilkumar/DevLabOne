### Car Detection and Manufacturer Classification
- Status: Approved for MVP
- Date: 2026-07-10
- Submitted by: CEO
- Target customer: Security monitoring teams, parking operators, building administrators, vehicle inspection teams, and DevLab-One demo users
- Problem: Existing project is incomplete and needs an end-to-end AI application that detects cars, classifies manufacturer, and shows results in image and live camera UI.
- Proposed solution: Use YOLO for car detection, crop detected cars, classify each crop with a ResNet50 transfer learning model trained on five manufacturers, expose Django APIs, and build Angular upload/live monitoring UI.
- Revenue path: AI computer vision demo, paid custom surveillance/parking/inspection solution, future SaaS or managed AI monitoring product.
- Required AI agents: CEO, Business Analyst, Project Manager, Scrum Master, Solution Architect, ML Engineer, Backend, Frontend, UX Designer, QA, Security, DevOps, Documentation
- Score: CEO-selected active project
- Recommendation: Proceed with discovery, dataset audit, model training, API, UI, and QA phases.
- CEO decision: Approved as active project.
- Next action: Confirm five manufacturer names and dataset location.
